Over the past decade, bicycle-sharing systems have been growing in number and popularity in cities across the world. Bicycle-sharing systems allow users to rent bicycles on a very short-term basis for a price. This allows people to borrow a bike from point A and return it at point B, though they can also return it to the same location if they'd like to just go for a ride. Regardless, each bike can serve several users per day.

A/B Testing

A website that was launching a new page needed to know the effect of changing their layout on users using practical analysis and A/B Testing, On this project, I worked on many procedures to find an answer for the question as follows:

  • Probability distributions throughout the users database and how it was affected by the change
  • Hypothesis Testing
  • Regression Analysis
Finally, I reached out that there's not a real impact of the layout change so the suggested decision is to stay on the old layout.
